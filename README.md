# Ailes Global

Ailes Global is a premium, mobile-first visa automation platform for African entrepreneurs applying for Schengen visas.

## Stack

- Next.js 16 App Router
- TypeScript
- Tailwind CSS
- Lucide Icons
- Framer Motion
- Supabase-ready architecture
- TanStack Query and Zustand ready

## What's Included

- High-trust landing page hero for Global African Entrepreneurs
- Clean folder structure for components, lib, hooks, and app
- Placeholder /apply route for the future visa intake flow
- Shared UI helpers for building the dashboard and intake modules

## Project Structure

- app/ — pages and routes
- components/ — layout, UI, and feature modules
- lib/ — utilities and configuration
- hooks/ — reusable client hooks

## Run Locally

npm install
npm run dev

Then open http://localhost:3000.
